# Student Register Project

This is a full-stack student registration web app built with:

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL (cloud-hosted, e.g., PlanetScale)
- Deployment: Vercel

## Features

- Create, Read, Update, Delete (CRUD) student records
- Responsive frontend with form and student list
- Backend API routes for handling database operations
